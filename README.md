# Company Classification Analysis

EDINETから取得した有価証券報告書の【事業内容】テキストをベースに、SBERT（Sentence-BERT）を用いた自然言語処理（NLP）によって、企業を東証が定める「33業種」へ確率的に分類・分析するプロジェクトです。

業種分類では見えない、企業の注力している事業が数値的に可視化されている点がポイントです。

## 主な機能と特徴

- **EDINETスクレイピング:** 有価証券報告書の【事業内容】および【業績等の概要】セクションの自動抽出
- **SBERTによる埋め込み & 分類:** 事前学習済みSBERTモデルを用いた事業内容テキストのベクトル化、および東証33業種への確率的分類モデルの構築
- **可視化:** 分類された確率分布（業種間の類似度など）を基にした、企業の空間的な立ち位置の表示
